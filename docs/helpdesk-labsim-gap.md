# Getting closer to CompTIA A+ / TestOut LabSim

LearnForge cannot license TestOut LabSim. We built the strongest **in-app** help-desk practice we can, and point learners to **free** tools for what we cannot simulate.

## What TestOut LabSim has that we approximate

| LabSim feature | LearnForge approach |
|----------------|---------------------|
| Ticket queue / severity / escalate | **Help desk ticket queue** engine (`helpdesk-ticket-queue`) |
| CLI troubleshooting | Terminal labs (ipconfig, ping, DNS, accounts, printers) |
| Hardware assembly | PC build order + code diagnostics |
| Soft skills / security | Script-choice phishing, MFA, angry-user scenarios |
| Ports / protocols | Match-pairs lab |
| Full Windows GUI campus | **External:** Microsoft Learn |
| Routers / switches campus | **External:** Cisco Packet Tracer (NetAcad) |
| Extra browser A+ labs | **External:** LabEx A+ labs |
| Video curriculum | **External:** Professor Messer (free) |

## In-app: IT Support track (priority order)
